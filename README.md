# Food Recipes web application

A responsive web application in which allows users to to search for, view, and save recipes.

### Languages and tools used:
- Typescript
- ReactJS
- Tailwind
- React Select
- React Router
- Vitejs
- VSCode

#### All data was fetched from an open-source API: https://spoonacular.com/food-api 

Users are able to search recipes by recipe's name, diet type, meal type, food intolerances, cuisines, and ingredients.

Users are also able to save recipes into meal categories (e.g. breakfast, lunch, snacks, etc), and add ingredients in the grocery shopping list.

Each recipe display the following information:

- Name
- Picture
- Number of likes
- Author
- Summary
- Total time to prepare
- Ingredients list
- Preparation steps
- Nutritional information
